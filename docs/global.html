# Modal(ModalProps) → {JSX.Element}

Provides the structure of a modal with:

  • A header with the title of the latter,
  • An empty body which can be filled with the children prop,
  • A footer with a confirmation button and a button cancellation belonging to the form whose id corresponds to the value of the buttonsForm prop.
